## 学习路径（建议）

1) VM 配置模板化（规格/驱动/工具） → 2) 性能（vCPU/vNUMA/内存保留/磁盘/网络） → 3) 迁移与维护窗口 → 4) 备份/恢复 → 5) 安全

## 快速Checklist

- 规格模板与标签管理，合规镜像与硬件版本
- VMware Tools/驱动一致性、I/O 调优（Paravirtual/VMXNET3）
- 热迁移前检查（带宽/延迟/锁定）与维护窗口计划
- 应用一致性备份（VSS/应用钩子）与恢复演练
